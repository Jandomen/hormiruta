# Hormiruta

Hormiruta es una aplicación web para la gestión de rutas, gastos y alertas SOS. Permite a los usuarios registrar rutas, administrar gastos asociados, recibir alertas geolocalizadas y gestionar sus suscripciones a través de RevenueCat. La plataforma incluye autenticación segura mediante Google OAuth y soporte opcional de registro con email y contraseña. Está desarrollada con Next.js, TypeScript y MongoDB Atlas, y lista para desplegar en Vercel.
